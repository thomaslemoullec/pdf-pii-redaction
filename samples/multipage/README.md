# Multi-page sample packages

Four small (2–3 page) multi-document packages — bank statements, application forms,
proof-of-address, notary and income letters stitched together — used to exercise the
**multi-page / multi-document** path (`make seed` uploads these to
`gs://<bucket>/incoming-multi-page/`).

**Provenance:** these are **synthetic account-opening packages assembled from the public
RVL-CDIP research dataset** (scanned document images), *not* generated by this repo's
`scripts/generate_samples.py` and *not* real customer data. They're great test fixtures
for the vision pipeline, but note they carry RVL-CDIP's provenance/licensing — review that
before redistributing this repo publicly. The single-page samples in `../` (the parent
`samples/` folder) are fully synthetic, generated by `make samples`, and safe to share.
